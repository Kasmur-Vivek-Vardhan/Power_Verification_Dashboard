# 🚀 Quick Start Guide

Get your Power Verification Dashboard up and running in minutes!

## ⚡ Immediate Next Steps

### 1. Test Locally First (2 minutes)
Double-click `index.html` to open in your browser and verify everything works.

### 2. Set Up GitHub Pages (5 minutes)
1. Create GitHub account if needed: [github.com](https://github.com)
2. Create new repository: `power-verification-dashboard`
3. Upload all files from this folder
4. Enable GitHub Pages in Settings → Pages
5. Your site: `https://yourusername.github.io/power-verification-dashboard`

### 3. Upload Your First Data (1 minute)
1. Open your dashboard
2. Click "Admin Mode" 
3. Password: `PowerVerification2026!`
4. Paste your table data and save

## 🔧 Customization Options

### Change Admin Password
Edit `js/config.js`:
```javascript
ADMIN_PASSWORD: 'YourNewPassword2026!'
```

### Modify Colors
Edit `js/config.js`:
```javascript
CHART_COLORS: {
    primary: '#your-color',
    secondary: '#your-color'
}
```

### Add Company Logo
Edit `index.html` header section:
```html
<h1 class="logo">
    <img src="your-logo.png" alt="Logo">
    Your Dashboard Title
</h1>
```
